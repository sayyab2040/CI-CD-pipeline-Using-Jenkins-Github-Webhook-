# Frontend Module

This is the presentation layer of our 3-tier application.

## Technologies Used
* HTML5
* CSS3 (Custom properties, Flexbox, Animations, Glassmorphism)
* JavaScript (ES6, fetch API)
* Nginx (for containerization)

## Features
* Modern, responsive design
* Interactive tab switching
* Dynamic API URL resolution (works on localhost and EC2 public IPs without code changes)
* Loading states and visual feedback

## Local Development
To run this locally without Docker, simply open `index.html` in your web browser. Note that the backend must be running on `localhost:3000` for API calls to work.
